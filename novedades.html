Novedades

En esta página encontrarás las versiones de este sitio web.

Es cierto que no todos saben manejar GitHub, por lo que para muchas personas es complicado revisar los últimos lanzamientos, por lo que los pondré en esta página junto con enlaces a las releases que lanzo en GitHub.

Pero, ¿qué es GitHub? GitHub es una plataforma para desarrolladores, puedes seguirlos, ver sus programas y marcar los códigos que te gusten con estrellas, así como hacer bifurcaciones, algo así como una copia de los códigos de otras personas para colaborar.

¿Qué es una release? Una release es un lanzamiento de una versión de un software, pero como sé que muchos no saben ni siquiera que es GitHub o como funciona, hice esta página de novedades, para que no te quedes atrás con las mejoras.

Sin embargo, para aquellas personas más experimentadas pondré el enlace a las releases que haga.

Recuerda, no importa si no sabes usar GitHub, cada quien tiene sus fortalezas y debilidades Yo por ejemplo, no sé nada sobre las redes sociales de músicos o de pintores, pero eso no me hace menos listo o más listo, simplemente a mí se me da mejor esto, y está muy bien, justo por esto hago esta página de novedades, para que la gente que no entienda de los tecnicismos de GitHub no se quede sin conocer las actualizaciones de este sitio web.

Versiones

Versión 1.0.0

La primera versión 1.0.0, es la que probablemente viste cuando publiqué la página en Facebook.

  • Se hizo una página de inicio en dónde presento el sitio web, y te explico la funcionalidad del mismo.
  • Se hizo una página de aplicaciones En dónde he puesto algunas aplicaciones accesibles con descripciones y precios, junto con su enlace de la App Store.
  • Se ha puesto una página de juegos En donde se ponen varios juegos accesibles, al igual con descripciones, precios y enlaces de descarga.
  • Se incluyó una página de atajos Aquí pongo atajos que podrían ser útiles, pero sólo tengo uno que hice yo, si quieres que ponga un atajo creado por ti sólo envíame mensaje, y lo incluyo junto con el reconocimiento de que lo hiciste tú.
  • Se hicieron estilos para el sitio web, CSS hice estos estilos para aquellas personas que tengan restos visuales, que sea una experiencia más agradable de ver, también el objetivo es hacer que el sitio tenga alto contraste con letra un poco más grande, para que lo puedan visualizar bien
